RLP-004

Объект исследования: Кинематика дифференциальной тележки

Предмет исследования: Система интегро-дифференциальных уравнений движения

Математическая модель:

  • Уравнения в частных производных: \(\frac{\partial^2 \vec{r}}{\partial t^2} = f(\theta, \dot{\theta}, \int \tau dt)\)
  • Система связанных ОДУ: \(\begin{cases} m\ddot{x} = \tau_x - \beta\dot{x} + \lambda \int \dot{y} dt \\ m\ddot{y} = \tau_y - \beta\dot{y} - \lambda \int \dot{x} dt \end{cases}\)

Экспериментальная задача:

Дано: Тележка массой \(m = 2.5\text{кг}\) с начальными условиями:
\(x(0) = 0,\ \dot{x}(0) = 0.5\text{м/с}\)
\(y(0) = 1,\ \dot{y}(0) = 0\)
Параметры: \(\beta = 0.2\text{кг/с},\ \lambda = 0.15\text{кг/с}^2\)

Решение:

  1. Применение преобразования Лапласа к системе
  2. Получение интегральных представлений: \(x(t) = \mathcal{L}^{-1}\left[\frac{0.5(s + 0.3)}{s(s^2 + 0.4s + 0.45)}\right]\)
  3. Вычисление криволинейного интеграла: \(\oint_C (x\tau_y - y\tau_x) ds = \frac{m}{2}\frac{d}{dt}(x^2 + y^2)\)
  4. Аналитическое решение через специальные функции: \(y(x) = \frac{1}{3}x \cdot E\left(\frac{\pi}{4}, k\right) + \mathcal{O}(x^3)\)

Результаты:

  • Траектория описывается эллиптическими функциями Якоби
  • Предельная цикличность при \(t \to \infty\): \(\lim\limits_{t\to\infty} \frac{x}{y} = \sqrt{\frac{\lambda}{\beta}}\)
  • Энергетический диссипативный коэффициент: \(\eta = 1 - e^{-2\beta\pi/\lambda}\)
